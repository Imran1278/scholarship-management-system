# Scholarship Management System

An automated system to manage and determine scholarship eligibility for students based on multiple academic and extracurricular criteria.

##  Key Features
- **Eligibility Evaluation:** Calculates scholarship worth based on JEE Main marks and 12th-grade percentages.
- **Multi-Criteria Assessment:** Includes bonus points for sports achievements (National/State/District levels) and regional locality.
- **Financial Aid Logic:** Considers family background and caste for targeted financial assistance.
- **Data Persistence:** Automatically saves student records and their final scholarship details into a persistent text file (`K1609_47.txt`).
- **Interactive CLI:** A user-friendly console interface for data entry and result display.

##  Built With
- **Language:** C++
- **Concepts Used:** Object-Oriented Programming (OOP), File Handling (`fstream`), String Manipulation.

##  Project Structure
- `Scholarship-mgmt.cpp`: The main source code containing the class logic and scholarship algorithms.
- `README.md`: Project documentation.
- `K1609_47.txt`: (Auto-generated) Database file where student records are stored.
